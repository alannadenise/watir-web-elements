# watir-web-elements

## Requirements

* ruby 2

## Quick start

Clone the project:

```
git clone https://github.com/alannadenise/watir-web-elements.git && cd watir-web-elements
```

Get bundler:

```
sudo gem install bundler
```

Execute gemfile:

```
sudo bundle install
```

Run script:

```
ruby elements-datepicker.rb
```
